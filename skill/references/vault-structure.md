# Vault Structure

Recommended top-level folders:

```text
00-首页与导航
01-内容创作
02-AI工具与工作流
03-项目复盘
04-资料剪藏
05-每日简报
06-媒体素材
90-模板
99-归档
AI操作台
```

## Folder Intent

- `00-首页与导航`: home note, category navigation, and topic index entry points.
- `01-内容创作`: article drafts, video scripts, social posts, and publishable outlines.
- `02-AI工具与工作流`: AI tool notes, prompt workflows, automation records, and model experiments.
- `03-项目复盘`: project records, milestones, decisions, and postmortems.
- `04-资料剪藏`: clipped references, inbox material, source notes, and historical sync records.
- `05-每日简报`: daily summaries and periodic reports.
- `06-媒体素材`: images, screenshots, PDFs, videos, audio, and reusable assets.
- `90-模板`: note templates.
- `99-归档`: old, duplicate, or no-longer-active material.
- `AI操作台`: AI-generated outputs, backups, logs, and operator notes.

For real migrations, keep a move map and update wiki links after moving notes. For this template, only copy sample notes.
